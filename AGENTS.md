# 项目名称：Teacher Hub License Manager

## 项目定位

这是 Teacher Toolkit 的内部授权管理工具。

负责：
- 生成授权码
- 管理授权记录
- 查询与筛选授权记录
- 导出授权记录
- 管理导出设置
- 管理私钥导入与本地私钥配置
- 构建 Windows 安装包分发流程

不负责：
- 教师端业务模块
- 客户端验权页面
- 共享协议字段定义

## 技术栈

- Flutter
- SQLite
- Excel

## 平台优先级

本项目优先支持：

1. Windows
2. macOS
3. Android
4. iOS
5. 鸿蒙

当前阶段优先实现 Windows / macOS 桌面版。

## 关键规则

- 私钥不得提交到 git
- 私钥不得进入 `teacher_hub`
- 本项目必须依赖 `teacher_toolkit_license_protocol`
- 授权记录必须可追溯、可查询、可导出
- 导出目录属于应用级设置，必须放在 `lib/app/` 或 `lib/core/` 中统一管理
- 私钥导入与私钥本地保存路径属于应用级设置，必须放在 `lib/app/` 或 `lib/core/` 中统一管理
- 系统目录选择当前通过跨平台目录选择器实现，优先保证 Windows / macOS 体验
- 私钥文件导入当前通过跨平台文件选择器实现，优先保证 Windows / macOS 体验
- Android / iOS 的目录选择与文件选择能力可尝试通过同一插件覆盖
- 鸿蒙目录选择、文件导入兼容性当前未确认，如正式支持需单独验证后再更新文档
- 授权记录列表导出当前统一使用 `.xlsx`
- 所有长期保留的导出文件格式统一使用 `.xlsx`
- 所有导入功能都必须包含 `操作标记` 字段
- 所有导入功能都必须通过系统文件选择器选择文件
- 不得将手动输入文件路径作为正式导入交互入口长期保留
- 所有导入模板统一使用 `.xlsx`
- `操作标记` 允许值仅为 `I / U / D`
- 单次导入最多支持 `1000` 条数据
- 导入失败结果必须完整展示失败原因，不得只显示汇总数量
- 导入列数不得多于模板要求字段个数
- 导入文件名称允许扩写，但必须包含系统下载模板的名称
- 每个导入功能都必须有清晰的规则说明展示位置
- 每个导入功能都必须有清晰的字段说明展示位置
- 每个导入功能都必须提供配套模板下载入口
- 导入时必须基于唯一约束或业务主键判断：
  - `I`：目标记录必须不存在
  - `U`：目标记录必须已存在
  - `D`：目标记录必须已存在
- 所有导出功能都必须在最后一列附加 `操作标记`
- 导出时 `操作标记` 默认值统一写为 `I`
- 涉及文件或目录位置提示时，应提供“打开目录”动作
- 导入或导出结果如展示文件路径或目录信息，提示默认显示时长统一为 `5` 秒
- 若某类导入在当前阶段只允许 `I`，必须在对应模块文档中明确写出原因和限制
- Windows 发布版建议通过安装包分发，而不是直接要求用户手动拷贝整个 Release 目录
- 安装包脚本可以进入仓库，但私钥文件不得进入安装包默认内容
- 在 PowerShell 中读取本项目 UTF-8 文本文件时，优先使用 `Get-Content -Raw -Encoding UTF8`
- 不依赖 PowerShell 默认编码猜测来读取 `.dart`、`.md`、`.json`、`.yaml` 等文本文件
- 如遇中文内容异常，优先确认文件实际 UTF-8 内容，再决定是否重写或 patch
- 新增任何长期保留的功能时，都必须优先考虑操作流畅性
- 不得为了增加功能而明显牺牲首屏进入、页面切换、点击反馈、滚动顺滑度和高频操作体验
- 涉及授权记录列表、导入导出、筛选搜索、详情切换等高频场景时，应优先选择更轻量、更懒加载、更少阻塞的实现方式
- 所有长期保留功能中的日期选择、日期展示和日期相关文案，统一优先使用中文年月日格式
- 不得在面向用户的时间选择器、日期选择器、日期输入框、日期文本展示中默认使用英文月份、英文星期或英文日期文案
- 如使用系统或框架提供的日期选择组件，必须优先通过本地化配置保证其显示为中文
- 若某个平台组件暂时无法完整显示中文日期，必须在实现前主动说明影响范围与替代方案
