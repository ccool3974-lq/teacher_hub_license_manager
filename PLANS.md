# Teacher Hub License Manager 实施计划

## 当前实现状态：去版本化离线密钥重构

- 发码工具已改为生成 TTK3 离线登录密钥
- 已移除授权版本分层和功能点授权字段
- 授权记录继续保留应用版本号、绑定用户、用户编号、有效期、永久授权、首次激活截止、状态、操作人、备注、授权码和导入导出能力
- 数据库通过破坏式去版本化迁移移除 `tier` 列，并尽量复制保留已有记录
- 导入导出模板已删除“授权版本”列，新模板不再兼容旧列顺序
- `teacher_hub` 尚未在本计划中声明完成新协议接入，两端同步前不能混用新旧协议

## Phase 1：项目壳层与基础结构

- 建立目录骨架
- 完成根文档
- 建立应用入口与基础路由

## Phase 2：授权记录数据层

- 建立 `license_records` 表
- 完成 entity 与 repository

## Phase 3：授权生成业务层

- 接入共享协议库
- 完成 payload 构造
- 完成签名流程
- 生成 TTK3 离线登录密钥，不再生成不同授权版本或功能权限

## Phase 4：授权创建 UI

- 完成新建授权页面
- 生成授权码并落库
- 支持设置首次激活截止日期，并在表单、详情、导出中统一展示

## Phase 5：授权记录与导出

- 完成授权记录列表页
- 完成授权详情页
- 支持单条导出与列表导出
- 支持状态管理

## Phase 6：导出设置与导出体验

- 增加导出目录设置入口
- 支持系统目录选择
- 支持自定义导出目录与恢复默认目录
- 授权记录列表导出统一改为 `.xlsx`
- 导出字段补充创建时间、更新时间
- 导出后支持打开导出目录
- 导出成功提示停留时间统一调整为 5 秒

## Phase 7：Windows 安装包分发

- 提供 Inno Setup 安装脚本
- 统一从 Windows Release 目录构建安装包
- 创建桌面快捷方式与开始菜单入口
- 明确安装包不默认携带私钥文件
- 在 README 中补充打包与安装说明

## Phase 8：私钥导入与本地配置

- 增加私钥设置页面
- 支持通过系统文件选择器导入私钥文件
- 将导入后的私钥副本保存到应用本地设置目录
- 发码前可检测当前是否已配置私钥
- 在新建授权页和记录主页提供私钥设置入口

## Phase 9：批量导入与批量生码

- 支持导入已有授权记录 `.xlsx`
- 支持按授权编号进行导入覆盖或补录
- 支持导入待发码名单并批量生成授权码
- 提供批量生码模板下载
- 批量结果统一写入授权记录列表
