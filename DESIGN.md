# Teacher Hub License Manager 设计文档

## 1. 目标

`teacher_hub_license_manager` 是 Teacher Toolkit 的内部授权管理工具。

目标：
- 生成 TTK3 离线登录密钥
- 保存并管理发码记录
- 支持追溯、查询和导出

## 2. 架构

本项目采用三层结构：

- `data`
- `domain`
- `presentation`

## 3. 与其他项目关系

- `teacher_hub`
  - 客户端
  - 只持有公钥
- `teacher_hub_license_manager`
  - 内部发码工具
  - 持有私钥
- `teacher_toolkit_license_protocol`
  - 共享协议库

## 4. 离线密钥协议边界

- 发码工具只生成离线登录密钥，不做免费版、基础版、高级版分层
- 发码工具不按功能点授权，不维护 `tier` 或 `features`
- 离线密钥保留应用版本号、绑定用户、用户编号、有效期、永久授权、首次激活截止、状态、操作人、备注、授权码与导入导出能力
- 首次激活截止约束授权码首次使用窗口，永久授权也必须保留该字段
- `teacher_hub` 未同步接入新协议前，不能把新 TTK3 离线密钥写成已完成客户端接入

## 5. 批量能力边界

本项目支持两类批量处理：

### 5.1 导入已有授权记录

用途：
- 历史数据迁移
- 从外部备份文件恢复授权记录

规则：
- 以 `licenseId` 作为业务唯一键
- 允许按 `licenseId` 覆盖现有记录
- 导入文件必须包含原始授权码
- 页面必须提供“导入规则说明”和“导入字段说明”两个独立入口

### 5.2 批量生码导入

用途：
- 导入待发码名单
- 由系统批量生成授权码并写入授权记录表

规则：
- 导入文件不要求已有 `licenseId`
- 每一行由系统生成新的 `licenseId` 和授权码
- 导入结果直接进入授权记录列表
- 页面必须提供“导入规则说明”和“导入字段说明”两个独立入口
